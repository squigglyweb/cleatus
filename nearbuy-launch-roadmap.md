# The Local NearBuy — Launch Roadmap

A step-by-step guide to building The Local NearBuy from the ground up.

---

## Phase 1: Foundation (Weeks 1-2)

### 1. Export Logo Files
- [ ] Export NearBuy logo in: SVG, PNG, PNG @2x, White PNG, Favicon, Apple Touch (180x180)
- [ ] Export Square version for social profiles
- [ ] Export Horizontal version (if applicable)

### 2. Set Up Social Accounts
- [ ] Create/optimize Facebook page: The Local NearBuy
- [ ] Create/optimize Instagram account
- [ ] Create/optimize Google Business Profile (try as service area business)
- [ ] Create LinkedIn profile (optional)
- [ ] Set profile photos, bio, link to placeholder site

### 3. Brand Assets
- [ ] NearBuy red color code: #d32f2f
- [ ] Font choices (match existing branding)
- [ ] Template for social posts (Canva) — use the prompt provided

---

## Phase 2: Content Engine (Weeks 3-6)

### 4. Build 30-Day Content Calendar
- [ ] Finalize content pillars
- [ ] Write first 30 post captions
- [ ] Schedule posts in Meta Business Suite
- [ ] Go live with daily posting

### 5. Visit & Document Local Businesses
- [ ] Visit 1-2 businesses per week
- [ ] Take 3-5 photos at each
- [ ] Write your review/caption
- [ ] Post and tag the business

---

## Phase 3: Website Build (Weeks 4-10)

### 6. Set Up WordPress + Extra Theme
- [ ] Get hosting (if not already)
- [ ] Install WordPress
- [ ] Install Extra theme + Elegant Themes subscription
- [ ] Set up basic site structure

### 7. Build Homepage
- [ ] Hero section with mission + meal counter
- [ ] Featured business of the week
- [ ] Category quick-links
- [ ] Newsletter signup
- [ ] Current EDDM campaign preview

### 8. Build Directory Section
- [ ] Set up categories (Coffee, Pizza, Date Night, etc.)
- [ ] Create Business Page template
- [ ] Add your first 10 business write-ups
- [ ] Connect Google Reviews embedding
- [ ] Set up reader rating system

### 9. Build Business Portal (Advertiser Login)
- [ ] Set up member login system
- [ ] Dashboard: redemption stats, QR codes, download assets
- [ ] Inquiry form for new advertisers

### 10. Build Ad Builder Interface
- [ ] Create postcard grid visual
- [ ] Tier selection (Small/Medium/Large)
- [ ] Click-to-select available spots
- [ ] Price + meal calculator
- [ ] Custom QR code generation

### 11. For Businesses Page
- [ ] EDDM pricing page
- [ ] How it works explanation
- [ ] Meal impact explanation
- [ ] Inquiry/booking form

---

## Phase 4: Newsletter (Weeks 6-12)

### 12. Set Up MailerLite
- [ ] Create account (free up to 1,000)
- [ ] Design newsletter template (match NearBuy brand)
- [ ] Create signup forms for site
- [ ] Set up welcome sequence

### 13. Publish First Issues
- [ ] Issue 1: Welcome + mission
- [ ] Issue 2: Featured businesses
- [ ] Build subscriber base from social + site

---

## Phase 5: EDDM Launch (Weeks 10-16)

### 14. First EDDM Campaign
- [ ] Sell initial ad spaces (target: fill most spots)
- [ ] Design postcard layout
- [ ] Collect business ads + QR codes
- [ ] Print and mail via EDDM
- [ ] Track redemptions

### 15. Measure & Optimize
- [ ] Redemption rate report
- [ ] Business feedback
- [ ] Adjust pricing/tiers if needed

---

## Phase 6: Events (Q2+)

### 16. First Meal-Packing Event
- [ ] Find venue (church, community center)
- [ ] Invite local businesses (sponsor swag)
- [ ] Invite local officials
- [ ] Promote across all channels
- [ ] Document for content

## Content Pillars

1. **Coffee Corner** — Weekly coffee shop spotlight
2. **Pizza Files** — Local pizza rankings/reviews
3. **Date Night Worthy** — Restaurants for special occasions
4. **The Sweet Spot** — Ice cream, bakeries, desserts
5. **Home Heroes** — Plumbers, HVAC, electricians
6. **Smile Squad** — Dentists, orthodontists
7. **Hidden Gems** — Places most people don't know about
8. **Local Love** — General celebration of great local businesses
9. **Meals4Good** — Cause marketing woven throughout
