## Offer Marketing Strategy (May 8, 2026)

### Key Insight
Offers can be HIDDEN from the website during print campaigns. This creates value for printed postcards vs website-only listings.

### How It Works
- Website offers: Always visible (included in membership)
- Print campaigns: Hide offer on website, only accessible via QR code on printed postcard
- After campaign: Offer goes back live on website

### Business Value
- Trackable ROI: "47 redemptions from postcard vs 12 from website"
- Proves print ROI vs digital
- Forces people to use the physical postcard = verifiable traffic source

### Pricing
- Website listing: Included in Pro/Pro+ membership
- Print campaigns: $150/campaign (extra)
- Hidden offer feature: The differentiator that proves print value
